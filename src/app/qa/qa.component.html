What is Sierra?

Sierra is a Firefox browser extension. It's a plugin that allows you to take control of your own data by selling it.

What is it's purpose?

When you use Firefox, you automatically create valuable data: how you browse is actually very interesting. Companies like Google when you use Chrome take this data and use it for multiple reasons, like improving the Chrome browser itself.
